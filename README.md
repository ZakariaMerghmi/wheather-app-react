# 🌦️ Weather App - React + Tailwind

A modern, responsive **weather app** built with **React** and **Tailwind CSS**.  
Originally developed using just HTML, CSS, and vanilla JS – this new version is now refactored and upgraded into a professional component-based structure using React.

## 🚀 Features

- 🔍 Search for weather by city name
- 🌡️ Dynamic weather data using JSON API
- 🌤️ Auto-updating icons based on weather description
- 📸 Error handling with custom "city not found" message and image
- 🎨 Clean UI with TailwindCSS

## 🛠️ Tech Stack

- React (Functional Components)
- Tailwind CSS
- Weather API (e.g. WeatherStack or similar)
